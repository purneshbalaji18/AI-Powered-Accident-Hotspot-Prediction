# AI-Powered Accident Hotspot Prediction

This project aims to predict accident hotspots using AI and machine learning techniques.

## Getting Started

### Prerequisites

- Python 3.8+
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd "AI-Powered Accident Hotspot Prediction"
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
